This is the documentation of the official Python module for the GTA Orange dedicated server. It allows you to code your resources in Python instead of LUA, allowing you to use the advantages of (C)Python, like the feature-rich built-in implementations and the endless list of packages contributed by the community. And once you tried it and saw some meaningful and beautiful error tracebacks, there is no way back to LUA - trust us! :-D

To get started, we recommend you to read the tutorial series. It teaches you how to use the module and its API as fast as possible. Of course, the reference can also be very helpful, but please note that the API is completely documented in-file with Python docstrings. So, if you use some kind of IDE, it may be faster to just have a look at the source code.
