Library Reference vehicle trigger

Description:

Triggers an event for all vehicles.

Reference:

GTAOrange.vehicle.trigger(event, *args)

Arguments:

  1. event (string) - event name
  2. *args (*args) - arguments

Return value:

Nothing.
